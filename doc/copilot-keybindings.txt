==============================================================================
1. OVERVIEW                                     *copilot-keybindings-overview*

This configuration uses custom keybindings to avoid conflicts with Tab
completion and provide easy access to Copilot features from both insert and
normal modes.

==============================================================================
2. ACCEPTING SUGGESTIONS                          *copilot-keybindings-accept*

Insert mode:
    <C-CR>      Accept the current suggestion (Ctrl+Enter)
    <C-j>       Alternative way to accept (Ctrl+J)

Note: Tab completion is disabled to avoid conflicts with other plugins.

==============================================================================
3. NAVIGATION                                   *copilot-keybindings-navigate*

Insert mode:
    <M-]>       Next suggestion (Alt+])
    <M-[>       Previous suggestion (Alt+[)
    <C-e>       Dismiss/cancel current suggestion

==============================================================================
4. PANEL ACCESS                                    *copilot-keybindings-panel*

Normal mode:
    <leader>cp  Open Copilot panel

Insert mode:
    <C-\><C-p>  Open Copilot panel (exits insert mode)

The panel shows multiple suggestions and additional context.

==============================================================================
5. COMMANDS                                     *copilot-keybindings-commands*

Normal mode:
    <leader>cs  Trigger new suggestion
    <leader>cd  Disable Copilot
    <leader>ce  Enable Copilot
    <leader>cr  Restart Copilot

To view this help again, use the command `:help copilot-keybindings`
